# .system-config

bash、zsh、vim 配置集合

## 1、使用方法

先将仓库clone到`$HOME`目录下
```
git clone https://github.com/G-dab/.system-config.git
```

### 1.1 Bash

```
~/.system-config/bin/install_bash.sh
```

### 1.2、Zsh
```
~/.system-config/bin/install_oh_my_zsh.sh
zsh ~/.system-config/bin/install_oh_my_zsh
```
